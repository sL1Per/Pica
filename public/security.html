Security

Passphrase, recovery code, and encryption-key rotation.

Change passphrase

The passphrase encrypts all data and backups. Keep it safe — without it, your data can only be recovered with the recovery code below.

If you unlocked with a recovery code, leave this blank.

At least 12 characters. A passphrase of four random words works well.

Passphrases don't match.

Recovery code

One-time code that can unlock your data if you forget the passphrase. Shown once — write it down and keep it somewhere safe (offline).

A recovery code is currently set. Generating a new one invalidates the old one.

Rotate encryption key

        
Generates a new key and re-encrypts all data. Use only after a suspected compromise. Old backups will no longer restore. The recovery code is cleared. Pica must restart.
