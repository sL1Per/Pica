# Guide screenshots

These PNGs illustrate the User and Admin guides
([`../user-guide.md`](../user-guide.md), [`../admin-guide.md`](../admin-guide.md)).

They are **illustrative**, captured from a throwaway instance in the
default Slate / light theme at en-US, desktop width (1440×900). Pica's UI
changes often and there is no automated screenshot regeneration (the
project has zero npm dependencies and does not ship Playwright), so a shot
may lag the live UI by a release or two — treat them as a guide to the
shape of each screen, not a pixel-exact reference.

To re-capture: stand up a throwaway seeded instance on a spare port with
its own data directory, drive it with a browser-automation tool, and save
over these files. The live install must never be used for this.

The sample data (company "Northwind Coffee", employees Ana / Bruno /
Clara / Diogo) is fictional.
